# 1. 基本操作

| コマンド | 説明 | サンプル |
|---|---|---|
| `pause` | 指定秒待機 | `pause 5` |
| `end` | マクロ終了 | `end` |
| `messagebox` | メッセージ表示 | `messagebox '完了' 'INFO'` |
| `beep` | ビープ音 | `beep` |

### 例

```ttl
messagebox '処理開始' 'INFO'
pause 3
messagebox '処理終了' 'INFO'
end
```

# 2. 接続系コマンド

## connect

サーバへ接続する。

### 構文

```ttl
connect 'hostname:22 /ssh /2 /auth=password /user=username /passwd=password'
```

### 例

```ttl
connect '192.168.1.100:22 /ssh /auth=password /user=test /passwd=test123'
```

### 補足

- SSH接続
- パスワード認証
- 秘密鍵認証も可能

---

## disconnect

接続終了。

```ttl
disconnect
```

# 3. 送信系コマンド

## sendln

改行付き送信。

### 例

```ttl
sendln 'ls -l'
```

実際の送信内容：

```bash
ls -l
```

---

## send

改行なし送信。

### 例

```ttl
send 'cd /tmp'
sendln ''
```

---

## sendfile

ファイル送信。

```ttl
sendfile 'C:\temp\test.txt'
```

---

## scpsend

SCP送信。

```ttl
scpsend 'C:\work\data.txt' '/tmp/'
```

---

# 4. 待機系コマンド

## wait

文字列待機。

### 例

```ttl
wait '$'
```

プロンプトが表示されるまで待機。

---

## waitln

行単位待機。

```ttl
waitln 'login'
```

---

## waitregex

正規表現待機。

```ttl
waitregex '.*error.*'
```

---

## timeout

タイムアウト秒数。

```ttl
timeout = 30
```

```ttl

```

# 6. 文字列操作

## strconcat

文字列結合。

```ttl
strconcat strvar 'abc'
```

---

## strlen

文字列長取得。

```ttl
strlen strvar
```

結果は：

```ttl
result
```

に格納される。

---

## sprintf2

文字列フォーマット。

```ttl
sprintf2 str '%s_%d.log' 'test' 1
```

結果：

```text
test_1.log
```

# 7. 制御構文

## if

条件分岐。

```ttl
if result = 1 then
    messagebox '成功' 'INFO'
endif
```

---

## goto

ラベルジャンプ。

```ttl
goto ERROR

:ERROR
messagebox 'エラー'
```

---

## for

ループ。

```ttl
for i 1 10
    sendln 'date'
next
```

---

## while

繰り返し。

```ttl
while i < 10
    i = i + 1
endwhile
```

## logclose

## 実務例：ログ取得

```ttl
logopen 'C:\log\evidence.log' 0 0

sendln 'pwd'
wait '$'

sendln 'ls -l'
wait '$'

logclose
```

# 9. 実務サンプル

## Linuxサーバ接続 → コマンド実行 → ログ取得

```ttl
timeout = 30

connect '192.168.1.100:22 /ssh /auth=password /user=test /passwd=test123'

wait '$'

logopen 'C:\temp\evidence.log' 0 0

sendln 'pwd'
wait '$'

sendln 'cd /var/log'
wait '$'

sendln 'ls -ltr'
wait '$'

logclose

disconnect

messagebox '完了しました' 'INFO'
end
```
